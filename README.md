# Wallpaper Website

## Project Overview
This project is designed to provide users with a comprehensive platform to view and download high-quality wallpapers.

## Features
- **User Authentication**: Users can sign up and log in to save their favorite wallpapers.
- **Browse and Search**: Users can browse through a collection of wallpapers or search for specific types.
- **High-Quality Downloads**: Users can download wallpapers in various resolutions.

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/lievaian41/LIEVA.git
   ```
2. Navigate to the project directory:
   ```
   cd LIEVA
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the application:
   ```
   npm start
   ```
